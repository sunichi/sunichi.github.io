buf的空间大小为0x400，read()的读入长度也为0x400。进入echo()函数继续查找。

字符串s2长度为0x10，内容从buf进行拷贝，当遇到\x00时特，停止拷贝。栈布局如下

size data
0x10 s2
0x08 saved ebp
0x08 return address
0x400 buf
