0x00 简介

AddressSanitizer(ASan)是一个内存错误检测工具，主要能够检测内存的非法读写、UAF等。

ASan通过Shadow Byte来记录内存空间的状态，每8字节有9个状态，分别标记从低到高Good Byte的数量。全8字节无问题为0，前7～1字节无问题分别为7～1，全部有问题标记为-1。Shadow Byte的位置通过如下计算方式得到：
